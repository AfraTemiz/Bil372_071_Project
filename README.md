# SchoolManagement
## Kurulum
node.js version 18 or higher


Sadece ilk çalıştırmada
```
server dosyası içinde
$ npm install
client dosyası içinde
$ npm install
```
```
çalıştırmak için önce server dosyası içinde
$ npm start
daha sonra client dosyası içinde
$ npm start
komutlarını çalıştırınız.
```
